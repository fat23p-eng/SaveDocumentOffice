ที่ตรวจสอบจากภาพ:
=======
กลุ่มทั่วไปทั้งหมด (ฝ่ายบริหาร, กลุ่มจัดตั้งฯ, กลุ่มส่งเสริมสหกรณ์ 1-6 ฯลฯ) → สิทธิ์ ทั่วไป ✅
งานนายทะเบียนสหกรณ์ และ งานสำนักงาน → สิทธิ์ ทั่วไป เหมือนกัน ✅ (ถูกต้องแล้ว ไม่ต้องตั้งเป็น "ดูทุกกลุ่ม") เพราะใน v3 ผมย้าย logic การ "เห็นได้ทุกคน" ของ 2 กลุ่มนี้ไปฝังไว้ในโค้ดที่ ALWAYS_VISIBLE_GROUPS แทนแล้ว ไม่ต้องพึ่งคอลัมน์สิทธิ์อีกต่อไป
admin → สิทธิ์ ผู้ดูแลระบบ ✅

ชื่อกลุ่มในชีตตรงกับที่ฝังไว้ในโค้ดเป๊ะ (งานนายทะเบียนสหกรณ์, งานสำนักงาน) ดังนั้นเรื่องตัวสะกด/เว้นวรรคไม่มีปัญหาแน่นอน
เหลือแค่เช็ค 2 อย่าง

แทนที่เนื้อหา Code.gs ด้วย Code_v3.gs และ Index.html ด้วย Index_v3.html แล้วหรือยัง (ถ้ายัง ลิงก์อยู่ในข้อความก่อนหน้า)
Deploy > Manage deployments > ✏️ > New version > Deploy อีกครั้ง (ขั้นตอนเดิมที่เคยทำสำเร็จมาก่อน)

ลองทดสอบล็อกอินด้วยกลุ่มทั่วไปสักกลุ่ม (เช่น ฝ่ายบริหารทั่วไป) แล้วเช็คในช่อง "กลุ่มที่อัพโหลด" ว่ามีตัวเลือกครบ 3 อย่าง: ของฉัน / งานนายทะเบียนสหกรณ์ / งานสำนักงาน ไหมครับ ถ้าได้แบบนั้นคือใช้งานได้ถูกต้องตามที่ต้องการแล้ว
